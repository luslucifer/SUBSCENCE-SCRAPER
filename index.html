Movie Subtitles API Documentation

Overview

The Movie Subtitles API allows users to access and download subtitle files for movies or TV series using either IMDb or TMDb IDs.

Endpoints

List Movie or TV Series Subtitles

Endpoint: GET /sub/<id>

Parameters:

  • id (required): The ID of the movie or TV series. This can be either an IMDb ID or a TMDb ID.
  • ss (optional): The season number if the ID corresponds to a TV series.
  • ep (optional): The episode number if the ID corresponds to a TV series.

Example:

GET /sub/tt1234567?ss=1&ep=5

Download Subtitle

Endpoint: GET /x/<path:url>

Parameters:

  • url (required): The URL-encoded path to the subtitle file.

Example:

GET /x/example_url.srt

Error Handling

  • 404 Not Found: If the requested movie or TV series, or subtitle file is not found.
  • 500 Internal Server Error: If there is an unexpected error while processing the request.

Rate Limiting

There is currently no rate limiting enforced on this API.

Authentication

This API does not require authentication.

Notes

  • Subtitle files are provided in the SubRip (.srt) format.
  • Use appropriate tools to handle subtitle file downloads and processing.
